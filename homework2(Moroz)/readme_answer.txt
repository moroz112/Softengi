Решение:

Метод Jquery each работает таким образом что проходиться по
всем обьктам выбраных Jquery селектором и применяет к ним callback
функцию. В данном случае как аргумент в callback передаеться индекс обьекта
с набора jQuery. Селектор такого вида $(div), где div - индекс не проходит проверки
в самой библиотеке Jquery, но обьект все же создаеться. Нам нужно
обратиться к каждому обьекту с выборки и применить к нему метод css.
Обратиться с помощью $(this) - что указывает на конкретный элемент из всех
элементов выбраных селектором $('div').

Второй вариант решения это передать в callback функцию второй параметр, что будет
указывать уже на конкретный обьект с набора обьектов.
